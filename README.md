## A graphQL server based on Express, SQLite and Relay connections 

### Installation

```
npm install
```

### Create database

```
sqlite3 db.sqlite3 < scripts/createdb.sql
```

### Start server

```
npm start
```

### Start GraphiQL tool

```
http://localhost:3000/graphql
```
